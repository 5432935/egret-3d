Egret3D 3.1.1 Release Note
===============================

最近更新时间：2016年6月1日

欢迎您使用Egret3D


## 概述
在Egret3D 3.1.1版本中，修复了3.1.0版本中相应的Bug,增加了部分功能

#### 更新内容
* 增加材质球UV repat
* 增加渲染 RenderToTexture
* 增加View3D背景贴图
* 增加MatCap材质球系列

#### Bug列表
* 修复正交相机渲染错误bug
* 修复HUD宽高渲染不正常bug
* 修复材质不给贴图时渲染异常的bug
* 修复同一个物体多次渲染异常的bug

#### 工具
* 增加unity3d模型导出插件
